Pocket PC Planet App Manager

Welcome to the Pocket PC Planet App Manager homepage! Want to download and install apps, right on your Pocket PC? This project aims to let you do just that!

There are two different versions, and you can learn more about them below!

Which version is right for you?

Pocket PC Planet App Catalog (Released)

This is a lite alternative to the main app manager program (see below) that is designed for older, less powerful Pocket PCs.

Pocket PC Planet App Manager (Coming Soon!)

A powerful and feature-rich app manager program for Pocket PCs! Coming soon.
